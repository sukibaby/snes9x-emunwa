> **Maintenance fork notice**
>
> This repository is a maintenance continuation of the original `snes9x-emunwa` fork.
> The original project has been unmaintained for over two years and no longer builds on modern Linux systems.
> This repo exists only to keep the Linux build working and to provide an GitHub built pre-built binary.

LINUX x86_64 precompiled release:

You'll need to decompress the binary from its `xz` archive and set it executable with `chmod +x ./snes9x-gtk`

LINUX compile locally:

To build it, refer to the workflow at .github/workflows/release.yml.
It details the exact steps needed to build on Ubuntu Linux.
