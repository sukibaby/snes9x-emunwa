LINUX:

Run the latest binary from the Releases section, or build it yourself.

- You'll need to decompress the binary from its `xz` archive.
- To build it, refer to the workflow at .github/workflows/release.yml.
It details the exact steps needed to build on Ubuntu Linux.
- Only x86_64 releases are currently available, arm64 is TBD
